项目库

所有开源项目的集合。为开源项目做贡献可能会令人生畏。尤其是当你是初学者时。这就是为什么我们创建了这个对初学者友好并由我们积极维护的开源项目集合。看看我们在这里的项目，我们相信您会找到您可能感兴趣的东西。

项目库是对初学者友好的开源项目的集合。初学者贡献者找到要贡献的项目的地方。项目库是每个人都可以找到活跃的初学者友好开源项目以查看和贡献的地方。

项目库也是一个开源网站，它在 AGPL 3.0 下获得许可，供所有人查看和研究我们的代码。您可以查看和贡献我们的源代码。您可以帮助在我们的 Github 存储库上为该项目做出贡献。

项目库目前由Jst 多功能工作室提供支持，起源于一个两人团队，Jst Tan 和 Poetsec。 Jst Tan 是一名网络开发人员、故事作者、视频和 wiki 编辑，而 Poetsec 是一名系统管理员，主要关注 IT、shell 脚本、硬件黑客和事物的一般安全方面。

项目库的想法源于 Reddit 的一篇文章，其中有人提到很难找到对初学者友好的好的（并且积极维护的）项目来贡献。我们希望改变这种状况，因此开源软件的世界以及在您喜爱和使用的项目上的协作应该可供所有人使用和访问。

建立你的信心

编码可能既困难又令人生畏，但通过为开源项目做出贡献，您可以建立信心并扩展您的技能。当您帮助完成不同的项目时，您有机会比自己学习更多。

扩大您的投资组合

如果您是科技界的新手，如果没有强大的投资组合，可能很难找到工作。从事开源项目表现出热情和奉献精神，对潜在雇主很有吸引力，更不用说随之而来的个人满意度。

与他人一起了解详情

虽然互联网文章和视频是一种近乎无限的学习材料资源，但您可以从经验丰富的开发人员那里了解更多信息。为开源做出贡献让您有机会与这些人建立联系并向他们学习。

与社区一起创建

我们在日常生活中都使用开源软件，但您是否曾希望某个软件能做得更好？通过为这些项目做出贡献，您可以在社区中产生影响，帮助构建您喜爱的软件并在此过程中结识有趣的人。
